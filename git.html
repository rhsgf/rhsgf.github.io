1.创建git库
wujiayi:~ lanou3g$ cd desktop //进入桌面
wujiayi:desktop lanou3g$ cd git //进入新建git
wujiayi:git lanou3g$ git init //初始化git库
wujiayi:git lanou3g$ git config --global user.name "wujiayi" //设置git库的名字
wujiayi:git lanou3g$ git config --global user.email "wujiayi6874@163.com" //设置git库的邮箱
wujiayi:git lanou3g$ touch index.html //在git库中新建index.html
wujiayi:git lanou3g$ git add . //添加git库下所有文件
wujiayi:git lanou3g$ git commit -m "text" //上传到git库
wujiayi:git lanou3g$ git diff //查看修改内容
wujiayi:git lanou3g$ git add . //修改后再添加
wujiayi:git lanou3g$ git commit -m "text" //修改后再上传
2.新建分支及合并
wujiayi:git lanou3g$ git branch //查看当前分支
wujiayi:git lanou3g$ git branch css //新建名为css的分支
wujiayi:git lanou3g$ git branch //查看当前分支是否添加成功新分支
wujiayi:git lanou3g$ git checkout css //切换到css分支
wujiayi:git lanou3g$ cd css //进入css分支
wujiayi:git lanou3g$ mkdir dir1 //在css分支新建dir1文件夹
wujiayi:git lanou3g$ cd dir1 //进入dir1文件夹
wujiayi:dir1 lanou3g$ touch test.html //在dir1文件夹新建test.html
wujiayi:dir1 lanou3g$ git add . //添加dir1下所有文件
wujiayi:dir1 lanou3g$ git commit -m "添加" //上传dir1下所有文件到git库
wujiayi:dir1 lanou3g$ cd ../ //返回dir1的上一级
wujiayi:git lanou3g$ git checkout master //切换到master主线
wujiayi:git lanou3g$ git merge css //合并css分支到master主线上
3.上传GitHub
先新建一个repository，名为gittest（名字随意，最好英文）
wujiayi:git lanou3g$ git remote add origin https://github.com/rhsgf/gittest.git //复制
wujiayi:git lanou3g$ git push -u origin master //复制
后刷新GitHub即可。当修改添加新内容上传到git库后，直接输入：
wujiayi:git lanou3g$ git push master 上传修改后内容到GitHub
4.通过github.io虚拟服务器上传到GitHub
先新建一个repository，名为rhsgf.github.io（GitHub名字.github.io）
通过GitHub Desktop程序点击Clone a Repository
复制GitHub新建页面中https://github.com/rhsgf/rhsgf.github.io.git到URL，点击Clone
进入点击open this repository，把名为rhsgf.github.io的文件夹拖到编辑器中
内容修改增删后，返回GitHub Desktop程序
在Summary处写此次修改之后的项目名字，Description处写描述（可不写），点击Commit to master
点击上方Push origin，后点击Fetch origin，上传成功后刷新GitHub即可
